Systems architect. Former CFO, PE portfolio company. Former Senior Editor at XDA.

Before Rust, a decade automating finance systems and ERPs in Python while running the departments that used them. Inherited enough failed integrations and broken automation to know where systems fail when nobody's watching. Governance built in, not bolted on.

## What I'm Working On

**Code intelligence and parser infrastructure.** [perl-lsp](https://github.com/EffortlessMetrics/perl-lsp): Rust-native Perl language server with tree-sitter and pure Rust parsing. [adze](https://github.com/EffortlessMetrics/adze): Rust-native grammar toolchain with GLR-capable parsing and typed extraction. [tokmd](https://github.com/EffortlessMetrics/tokmd): code intelligence for humans, machines, and LLMs.

**Governed SDLC.** [Adversarial build loops](https://github.com/EffortlessMetrics/demo-swarm). [Composable CI sensors](https://github.com/EffortlessMetrics/covguard) compiled into a [merge surface](https://github.com/EffortlessMetrics/cockpitctl). Each sensor answers one question about the change and produces a receipt.

**Legacy and regulated data.** [COBOL copybooks](https://github.com/EffortlessMetrics/copybook-rs), [Outlook PST/OST](https://github.com/EffortlessMetrics/pst-rs), [healthcare HL7](https://github.com/EffortlessMetrics/hl7v2-rs). Every enterprise AI engagement starts with getting data out of systems built before the architect was born.
